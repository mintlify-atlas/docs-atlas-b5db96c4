# .NET Runtime API

The sdkgen .NET runtime is implemented in F# and provides server runtime components for hosting sdkgen APIs in .NET applications.

## Installation

Add the runtime project reference to your .NET application:

```xml
<ProjectReference Include="path/to/dotnet-runtime/runtime/runtime.fsproj" />
```

## Package Structure

The runtime consists of three main modules:

- `Sdkgen.Context` - Request context types and parsing
- `Sdkgen.Helpers` - Type decoding helpers and error types
- `Sdkgen.Runtime` - HTTP server implementation

## BaseApi

Abstract class that must be implemented to define your API.

### Abstract Members

#### ExecuteFunction

Execute an API function and write the result to JSON.

```fsharp
abstract member ExecuteFunction: Context * Utf8JsonWriter -> Task
```

<ParamField path="context" type="Context" required>
  Request context containing function name, args, and metadata
</ParamField>

<ParamField path="writer" type="Utf8JsonWriter" required>
  JSON writer to output the function result
</ParamField>

#### GetAstJson

Return the API's AST as a JSON string.

```fsharp
abstract member GetAstJson: unit -> string
```

<ResponseField name="result" type="string">
  AST JSON string
</ResponseField>

**Example Implementation:**

```fsharp
type MyApi() =
  inherit BaseApi()
  
  override this.GetAstJson() =
    // Return your API's AST JSON
    File.ReadAllText("api.json")
  
  override this.ExecuteFunction(ctx: Context, writer: Utf8JsonWriter) =
    task {
      match ctx.Name with
      | "getUserById" ->
        let id = ctx.Args.["id"].GetInt32()
        let user = getUserById id
        writer.WriteStartObject()
        writer.WriteNumber("id", user.Id)
        writer.WriteString("name", user.Name)
        writer.WriteEndObject()
      | _ ->
        raise (FatalException($"Unknown function: {ctx.Name}"))
    }
```

## Context

Request context containing all information about the API call.

### Type Definition

```fsharp
type Context = {
  Name: string
  Args: Dictionary<string, JsonElement>
  DeviceInfo: ContextDeviceInfo
  Extra: Dictionary<string, JsonElement>
  mutable Headers: IHeaderDictionary
  Id: string
  mutable Ip: string
}
```

### Fields

<ParamField path="Name" type="string">

</ParamField>

<ParamField path="Args" type="Dictionary<string, JsonElement>">

</ParamField>

<ParamField path="DeviceInfo" type="ContextDeviceInfo">
  Information about the client device
</ParamField>

<ParamField path="Extra" type="Dictionary<string, JsonElement>">

### ContextDeviceInfo

<ParamField path="Id" type="string">

**Example:**

```fsharp

  // ... fetch user
```

## Error Types

### SdkgenException

Base exception class for all sdkgen errors.

```fsharp
type SdkgenException =
  inherit Exception
  val Type: string
  new(type': string, message: string, inner: Exception)
```

<ParamField path="Type" type="string">
  Error type identifier
</ParamField>

### FatalException

Exception for fatal/unexpected errors.

```fsharp
type FatalException =
  inherit SdkgenException
  new(message: string)
  new(message: string, inner: Exception)
```

**Example:**

```fsharp
open Sdkgen.Helpers

// Throw a fatal error
raise (FatalException("Database connection failed"))

// Throw with inner exception
try
  // ... some operation
with
| ex -> raise (FatalException("Operation failed", ex))
```

#### decodeOptional


**Example:**

```fsharp
open Sdkgen.Helpers

let processRequest (ctx: Context) =
  // Decode arguments
  let userId = 
    ctx.Args.["userId"]
    |> decodeInt32 <| "args.userId"
  
  let email = 
    ctx.Args.["email"]
    |> decodeOptional decodeEmail <| "args.email"
  
  let tags = 
    ctx.Args.["tags"]
    |> decodeArray decodeString <| "args.tags"
  
  // Use decoded values
  match email with
  | Some e -> printfn $"Email: {e}"
  | None -> printfn "No email provided"
```

## Active Patterns

The helpers module provides active patterns for pattern matching on JSON elements:

- `(|UInt32|_|)` - Match uint32 values
- `(|Int32|_|)` - Match int32 values
- `(|String|_|)` - Match string values
- `(|Double|_|)` - Match double values
- `(|Decimal|_|)` - Match decimal values
- `(|BigInt|_|)` - Match bigint values
- `(|Uri|_|)` - Match URI values
- `(|Guid|_|)` - Match GUID values
- `(|DateTime|_|)` - Match datetime values
- `(|Date|_|)` - Match date values

**Example:**

```fsharp
open Sdkgen.Helpers

let processValue (json: JsonElement) =
  match json.ValueKind, json with
  | JsonValueKind.Number, Int32 value -> 
    printfn $"Got integer: {value}"
  | JsonValueKind.String, Guid value -> 
    printfn $"Got GUID: {value}"
  | _ -> 
    printfn "Unknown type"
```

## Health Check

The server provides a built-in health check endpoint:

**GET /**

Returns:
- Status 200 with `{"ok": true}` if healthy
- Status 500 with `{"ok": false}` if unhealthy

Custom health checks can be configured via the `OnHealthCheck` handler.

## Request/Response Flow

1. Client sends POST request with JSON body
2. Server parses JSON into `Context` object
3. Server calls `api.ExecuteFunction(context, writer)`
4. Implementation writes result to JSON writer
5. Server wraps result in response envelope:
   ```json
   {
     "result": { /* your data */ },
     "error": null,
     "duration": 0.123456,
     "host": "server-hostname"
   }
   ```
6. On error, response contains:
   ```json
   {
     "result": null,
     "error": {
       "type": "ErrorType",
       "message": "Error message"
     },
     "duration": 0.123456,
     "host": "server-hostname"
   }
   ```
