## Welcome to sdkgen

sdkgen is a comprehensive toolkit for designing and implementing rich, type-safe APIs with minimal developer effort. Define your API contract once using a custom DSL, and generate strongly-typed client and server code for multiple platforms automatically.

## Simple API Definition

Describe your API using sdkgen's intuitive DSL with strong typing:

```sdkgen
type Post {
  id: uuid
  title: string
  body: string
  createdAt: datetime
  author: {
    name: string
  }
}

fn getPost(id: uuid): Post?
fn listPosts(limit: uint, offset: uint): Post[]
fn createPost(title: string, body: string): Post
```

Generate type-safe server and client code:

```bash
sdkgen api.sdkgen -o api.ts -t typescript_nodeserver
```

## Complete Example

Here's a complete example showing type definitions, error handling, and function declarations:

```sdkgen
error NotFound
error InvalidArgument {
  argumentName: string
  reason: string
}

type User {
  id: uuid
  avatar: url?
  name: string
  type: enum {
    guest
    fullUser
    admin
  }
}

type Post {
  id: uuid
  title: string
  body: string
  createdAt: datetime
  author: User
}

@description Get a user by their unique identifier
@arg id The unique identifier of the user
@throws NotFound
fn getUser(id: uuid): User

@description Get posts by a specific user
@rest GET /users/{userId}/posts?{since}&{until}
fn getPostsByUser(
  userId: uuid,
  since: datetime?,
  until: datetime?
): Post[]
```

## Why sdkgen?

### Type Safety Everywhere

With sdkgen, your API contract is the single source of truth. Changes to the API definition are immediately reflected in all generated code, catching breaking changes at compile time rather than runtime.

### Minimal Boilerplate

Focus on implementing business logic, not writing serialization code, validation logic, or client libraries. sdkgen handles all the tedious details.

### Protocol Agnostic

While sdkgen uses its own efficient binary protocol by default, you can expose the same functions as REST endpoints with a single annotation. Get the best of both worlds.

### Built for Scale

From small projects to large-scale APIs, sdkgen scales with your needs. Support for imports allows you to split large API definitions across multiple files.
